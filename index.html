⚙️ Learning outcome 2: Apply Vue Framework

Components (reusable blocks)

<script> export default { props: ['title'] } </script>

View components (pages) go in views/ and are route‑linked.

Vue Router

npm install vue-router

router/index.js

import { createRouter, createWebHistory } from 'vue-router'
import Home from '../views/Home.vue'
const routes = [
  { path: '/', component: Home },
  { path: '/about', component: () => import('../views/About.vue') },
  { path: '/:pathMatch(.*)*', component: NotFound } // 404
]
export default createRouter({ history: createWebHistory(), routes })

Use <router-link> and <router-view>.

State management (Vuex 4)

npm install vuex@4
// store/index.js
import { createStore } from 'vuex'
export default createStore({
  state: { count: 0 },
  getters: { double: state => state.count*2 },
  mutations: { increment(state) { state.count++ } },
  actions: { incAsync({ commit }) { setTimeout(() => commit('increment'), 1000) } }
})

Use useStore() (Composition) or mapGetters (Options).

⌨️ Keyboard fundamentals

  • Computer keyboard – peripheral for text input.
  • Types by key arrangement: QWERTY (standard), AZERTY (French).
  • By connection: wired, wireless, virtual (software).

Parts of a keyboard

  • Alphanumeric keys – letters, numbers, symbols.
  • Numeric keypad – calculator style (NumLock).
  • Function keys – F1‑F12 (help, refresh, etc.).
  • Cursor‑control keys – arrows, Home, End, Insert, Del, PageUp/Down.
  • Special keys – Ctrl, Alt, Shift, Win, Print Screen, Pause, etc.

🖐️ Typing technique & ergonomics

  • Home row: left hand on A,S,D,F ; right on J,K,L,; .
  • Keep wrists straight, elbows at 90°, monitor at eye level.
  • Use feather touch, don’t look at keyboard.

Typing speed measurements

TermFormula
WPM (words/min)(total chars / 5) / time (min)
Net WPMWPM – (errors / time)
CPMcharacters per minute
KPMkeystrokes per minute

Average speed: ~40 WPM, good: 60‑70 WPM.
